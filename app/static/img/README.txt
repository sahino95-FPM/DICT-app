Logo FPM
=========

Placez votre logo FPM dans ce dossier avec le nom: logo_fpm.png

Recommandations:
- Format: PNG avec fond transparent
- Taille minimum: 200x200 pixels
- Taille recommandée: 500x500 pixels
- Poids: < 200 KB

Le logo sera affiché dans:
- Header de l'application (50px de hauteur)
- Exports PDF

Si aucun logo n'est présent, l'application fonctionnera quand même
mais l'espace du logo sera vide.
